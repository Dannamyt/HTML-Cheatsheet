HTML Tags

Tag Description
<article> To define a article
<p> To define a paragraph
<body> To define the main part of the html document
<style> To style a HTML document internallly
<div> To define a division in HTML document
<footer> To define the footer section of the html document

HTML Attributes

Tag Description

class Used to identify group of HTML elements when styling in CSS
